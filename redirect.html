Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an Element in the DOM. document.getElementById is one of many javaScript method to select an element in the DOM. There are few more, such as document.getElementByClassName, querySelector and so on. Must remember, it is not a HTML tag! It is a JavaScript function that helps interact with HTML elements.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText and textContent is -
innerHTML gets or sets both text and HTML inside an element.
innerText retrieves only the visible text, ignoring hidden content and HTML tags.
textContent gets all text, including hidden content, but without any HTML formatting.

Question-3: What is event delegation in the DOM?

Event delegation is when you attach a single event listener to a parent element instead of each child element. The event listener checks which child triggered the event by using event.target. It helps improve performance, especially when elements are dynamically added.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event triggered on an element "bubbles up" to its parent elements. This means the event is passed from the target element to the parent elements in the DOM hierarchy. It allows parent elements to handle events for their child elements.

to Create, we use the function called createElement, and then write inside a parenthesis what we want to add.
